# 量化研究面板知识库

## 项目定位
这是一个围绕沪深300 ETF（510300）构建的量化研究面板，适合新手理解从数据更新、单次回测、参数扫描到训练/测试验证与 Walk-forward 稳健性评估的完整流程。

## 面板功能说明
### 1. 单次回测
适合快速验证当前参数组合。用户会看到核心收益指标、净值曲线、价格信号图和交易记录。

### 2. 参数对比实验
适合一次性比较多组参数。重点看 annual_return、sharpe、max_drawdown 的组合，而不是只看单一收益。

### 3. 验证实验
- 训练 / 测试验证：先在训练集找最佳参数，再在测试集看样本外表现。
- Walk-forward 验证：通过滚动窗口重复训练和测试，更接近真实研究流程，也更能暴露过拟合。

### 4. 实验历史
每次实验都会写入历史记录，可用于回看参数、指标、稳定性评分、研究评分和窗口细节。

## 新手教学原则
1. 先解释按钮和页面作用，再建议执行。
2. 涉及收益时，必须同时解释回撤、Sharpe、样本外表现和过拟合风险。
3. 当用户要求“帮我操作”时，优先直接调用工具执行，而不是只让用户自己点按钮。
4. 对 Walk-forward 结果，重点解释正收益窗口占比、跑赢基线占比、参数一致性和研究总分。

## AI 助教能力边界
AI 助教可以：
- 查看当前面板配置
- 更新数据
- 运行单次回测
- 运行参数对比实验
- 运行训练/测试验证
- 运行 Walk-forward 验证
- 检索实验历史并展开详情

AI 助教不能：
- 替用户做确定性的投资承诺
- 在没有证据的情况下宣称策略一定有效
- 忽略样本外结果只谈收益

## 持久记忆策略
长对话时不保留全部原始上下文，而是依赖：
- 最近若干轮消息
- 会话摘要
- 实验历史结构化记录
- 本知识库片段检索
这样可以避免上下文不断膨胀。
