# 量化研究 Agent 知识库

## 项目定位
这是一个聚焦 **沪深300横截面因子发掘** 的量化 Agent 系统，核心能力包括因子假设生成、因子评估、多轮进化搜索与经验沉淀。同时兼容单资产策略回测与稳健性验证。

## 核心能力（因子发掘主线）
### 1. 因子假设生成
根据研究方向自动生成候选因子，包含表达式树、族匹配、新颖性评分和探索奖励。

### 2. 因子进化搜索
多轮自主搜索闭环：每轮生成候选 → 评估 → 变异/淘汰 → 保留最优，直到收敛或达到轮次上限。

### 3. 因子评估与决策
对单个因子执行完整评估流水线，产出 Rank IC、ICIR、覆盖率、新颖性等指标，并给出 approve / observe / reject 决策。

### 4. 因子经验记忆
已验证的因子结论会被持久化到经验库，后续搜索可参考已有经验避免重复探索。

## 兼容能力（单资产回测）
### 1. 单次回测
快速验证当前参数组合，输出核心收益指标与交易记录。

### 2. 参数对比实验
一次性比较多组参数，重点看 annual_return、sharpe、max_drawdown 的组合。

### 3. 验证实验
- 训练 / 测试验证：先在训练集找最佳参数，再在测试集看样本外表现。
- Walk-forward 验证：通过滚动窗口重复训练和测试，更接近真实研究流程。

### 4. 实验历史
每次实验都会写入历史记录，可用于回看参数、指标、稳定性评分和研究评分。

## AI 助教能力边界
AI 助教可以：
- 生成因子假设与候选表达式
- 运行多轮因子进化搜索
- 评估单因子并给出决策建议
- 查看因子经验记忆与历史
- 更新横截面数据 / 单资产数据
- 运行单资产回测、参数对比、训练测试验证、Walk-forward 验证
- 检索实验历史并展开详情

AI 助教不能：
- 替用户做确定性的投资承诺
- 在没有证据的情况下宣称策略或因子一定有效
- 忽略样本外结果只谈收益
- 使用单资产代理横截面数据产出正式因子研究结论

## 持久记忆策略
长对话时不保留全部原始上下文，而是依赖：
- 最近若干轮消息
- 会话摘要
- 实验历史与因子经验结构化记录
- 本知识库片段检索
这样可以避免上下文不断膨胀。
