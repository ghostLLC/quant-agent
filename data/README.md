# 数据说明

本目录存放项目所需的行情与横截面数据。

## 核心数据（因子发掘主线）

| 文件 | 说明 |
|------|------|
| `hs300_cross_section.csv` | **主数据**——沪深300 横截面面板，因子发掘 / 进化搜索 / 评估均依赖此文件 |

### 横截面 CSV 字段要求

必须包含：
- `date` — 交易日期
- `asset` — 资产标识
- `close` — 收盘价
- `volume` — 成交量
- `industry` — 行业分类（可选，建议提供）
- `market_cap` — 总市值（可选，建议提供）

### 刷新方式

```bash
python refresh_data.py
```

## 兼容数据（单资产回测）

| 文件 | 说明 |
|------|------|
| `hs300_etf.csv` | 兼容——沪深300 ETF（510300）单资产日线，用于策略回测验证 |

### 单资产 CSV 字段要求

必须包含：`date`、`open`、`high`、`low`、`close`、`volume`

### 刷新方式

```bash
python fetch_hs300_etf.py
```

## 注意事项

- `date` 必须能被 pandas 解析，建议按日期升序排序，不要保留重复交易日
- 横截面数据是因子发掘 Agent 的核心输入，缺失时所有因子相关工具将无法运行
- 单资产数据仅用于回测兼容模式，不应作为因子发掘的数据源
